## Install & Preparation

1. Copy `.env.example` and rename to `.env`
2. Fill DB variable
3. Install dependencies

```bash
$ yarn
```

## Run

- Run server

```bash
$ yarn server:dev
```

- Run client

```bash
$ yarn client:dev
```

- Server url
